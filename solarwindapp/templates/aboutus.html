About Us

SolarWind is an AI-driven prediction system that determines whether Solar or Wind power is optimal based on environmental factors.

This project predicts the best power source (Solar/Wind) based on environmental factors such as wind speed, humidity, temperature, and wind direction.

How It Works

The system collects real-time environmental data and uses a trained Machine Learning model to predict whether Solar or Wind power is more efficient at a given moment.

Model Details

The model is a Random Forest Classifier trained on a dataset of 40,054 records, with an accuracy of 100%.


The model uses the following features:

  • --> Wind Speed
  • --> Humidity
  • --> Temperature
  • --> Wind Direction

API

The API endpoint is: /uploadsensordata/?nodeid=123&dcv=12&dci=200&acv=210

The API takes the following parameters:

  • --> nodeid: The ID of the node
  • --> dcv: The DC voltage reading
  • --> dci: The DC current reading
  • --> acv: The AC voltage reading
